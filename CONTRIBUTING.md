# Contributing to TASD Unified Framework

Thanks for your interest in contributing! TASD Framework is the mathematical unification of the Ψ_universe attractor with toroidal-modulated frequency-domain dynamics.

## Ways to Contribute

- **Bug reports** — open an issue describing the problem, your Python version, and steps to reproduce.
- **Feature requests** — open an issue with a clear description of the proposed feature and its relevance to TASD signal dynamics.
- **Code contributions** — fork the repo, make your changes, and submit a pull request.
- **Physics / mathematical feedback** — if you spot issues with the Lyapunov proof, TASD equation, or EAST validation, open an issue or discussion.

## Guidelines

1. **Python code** — keep dependencies minimal. NumPy and SciPy are acceptable; heavy frameworks should be justified.
2. **Reproducibility** — all examples and validations should be deterministic or clearly document their data sources.
3. **Test before submitting** — verify your changes run cleanly with both `tasd_framework.py` and `tasd_example.py`.
4. **Respect the framework** — TASD Framework unifies the Ψ_universe attractor with toroidal dynamics. Keep that theoretical framing in any documentation.

## Pull Request Process

1. Fork and create a feature branch (`git checkout -b feature/my-feature`).
2. Make your changes.
3. Test thoroughly.
4. Submit a PR with a clear description of what changed and why.
